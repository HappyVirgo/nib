### Keyboard shortcuts

| S.No. | Keyboard Shortcut      | Description  |
| ----- | ---------------------- | ------------ |
| 1     | Cmd-Alt-7 / Ctrl-Alt-7 | ordered list |
| 2     | Cmd-Alt-8 / Ctrl-Alt-8 | bullet list  |
| 3     | tab                    | indent       |
| 4     | shift+tab              | outdent      |
