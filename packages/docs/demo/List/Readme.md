### Keyboard shortcuts

| S.No. | Keyboard Shortcut          | Description  |
| ----- | -------------------------- | ------------ |
| 1     | Cmd-Shift-7 / Ctrl-Shift-7 | ordered list |
| 2     | Cmd-Shift-8 / Ctrl-Shift-8 | bullet list  |
| 3     | Tab                        | indent       |
| 4     | Shift-Tab                  | outdent      |
