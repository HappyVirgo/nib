### Keyboard shortcuts

| S.No. | Keyboard Shortcut          | Description    |
| ----- | -------------------------- | -------------- |
| 1     | cmd-b / ctrl-b             | bold           |
| 2     | cmd-i / ctrl-i             | italic         |
| 3     | cmd-u / ctrl-u             | underline      |
| 4     | cmd-shift-u / ctrl-shift-u | strike through |
| 5     | cmd-shift-m / ctrl-shift-m | code           |
