# @editr/ui

---

## These are reusable UI components for edit library.

---

## Components

- Button
- Select

---

## Setup

Either `npm` or `yarn` commands can be used.

```
npm i @editr/ui --save
```

### Writing editor component:

```
import React from "react";
import { Button } from "@editr/ui";

const MyEditor = () => (
  <Button>Click Me</Button>
);
```
